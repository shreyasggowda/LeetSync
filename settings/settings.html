LeetSync

Setup Guide

  1. Create a GitHub personal access token with repository write permissions.
  2. Open the extension popup and fill in your repository owner, repository name, and branch.
  3. Enable auto sync if you want every accepted solution pushed immediately.
  4. Open a LeetCode problem, submit a successful solution, and LeetSync will capture it.

This MVP uses a personal access token for GitHub instead of full OAuth to keep setup simple.
